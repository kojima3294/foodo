# Foodo（離乳食用献立アプリ）

昨年、うちにも子供が産まれたが離乳食のメニューを考えるのはとても大変でした。。。  
月齢にあったメニューを探したり、アレルギー食材を抜いたりと考える事が多いので、簡単に献立作成できないか？と考えこのWebアプリ開発に至りました。

## 機能

・最大1週間分（21食分）の献立が作成可能  
・月齢別（5-6ヶ月）（7-8ヶ月）（9-12ヶ月）に献立作成可能  
・アレルギー食材を献立から抜くことが可能
